# A1 · Unit 2 · Lesson 5 — Menschen beschreiben
### *Er ist groß.* — Describe people with adjectives

**Level:** A1 · **Unit:** 2 · **Lesson:** 5 of 5
**Capability:** *Describe yourself and others using simple adjectives.*
**Duration:** ~9 min · **New items:** 10 · **Grammar:** predicative adjectives (*sein* + adjective); *sehr / ein bisschen / nicht* modifiers
**Spiral:** *sein* (all persons), nationalities, *er/sie*, W-questions

---

## 1. Learning objective
Learner can:
- **describe** people with 8 common adjectives,
- **modify** with *sehr*, *ein bisschen*, *nicht*,
- **ask** *Wie ist er/sie?* and answer.

## 2. Success criteria
- [x] Produces *"Sie ist sehr nett. Er ist groß und lustig."*
- [x] Uses modifiers correctly
- [x] Feels "I can describe anyone in simple German"

## 5. Pattern Discovery — *sein* + adjective

**Notice:** All descriptions use the same pattern: **sein + adjective.** No extra words, no "is a tall" — just *Er ist groß.* Modifiers slide in before the adjective: *sehr nett, total lustig, eher klein.*

| Structure | Example |
|---|---|
| sein + adj | Er ist **groß**. |
| sein + modifier + adj | Sie ist **sehr nett**. |
| sein + nicht + adj | Er ist **nicht groß**. (= eher klein) |

**Why:** German adjectives after *sein* never change form — they stay in their base shape. (Unlike adjectives before nouns — that's for later.) Modifiers go between *sein* and the adjective.

## 6. Guided Practice (4 exercises)

### Ex 1 — Recognition — which adjective?
- **Prompt:** *"freundlich means?"* → ( funny ) ( **friendly** ) ( important )

### Ex 2 — Matching (opposites)
- **Match:** *groß · nett · lustig* ↔ *klein* · (no opposite) · (no opposite)
- *Note: not all adjectives have a clean opposite — highlight this.*

### Ex 3 — Fill-in
- **Prompt:** *"Sie ist ___ nett." (very)* → ( eher ) ( **sehr** ) ( nicht )

### Ex 4 — Micro-production
- **Prompt:** *Describe your friend: tall, very funny, friendly.* → **Er ist groß, total lustig und freundlich.**

## Common mistakes
| Mistake | Fix |
|---|---|
| *"Er ist ein groß"* | No article before adjectives — *Er ist groß.* |
| *"Sie ist nett sehr"* | Modifier before adjective — *sehr nett* |
| *groß* pronounced "gross" | *ß* = sharp "ss" — "grohss" |
