# A1 · Unit 2 · Lesson 4 — W-Fragen
### *Wer? Wie? Wo?* — Ask anything with W-questions

**Level:** A1 · **Unit:** 2 · **Lesson:** 4 of 5
**Capability:** *Use all five core W-questions to ask about people and things.*
**Duration:** ~9 min · **New items:** 8 · **Grammar:** W-questions review + *was* (new); question word order (verb second)
**Spiral:** *sein*, nationalities, *kommen aus*, *sprechen*, *heißen*

---

## 1. Learning objective
Learner can:
- **use** all five core W-words: *wer, wie, wo, woher, was*,
- **form** correct W-questions (W-word + verb + subject),
- **answer** each type appropriately.

## 2. Success criteria
- [x] Produces *Wer ist das? Wie heißt du? Wo wohnst du? Woher kommst du? Was ist das?*
- [x] Correct verb-second order in every question
- [x] Feels "I can ask any basic question in German"

## 4. Vocabulary — The five W-words + extras

| German | 🔊 | Meaning | Question example |
|---|---|---|---|
| **wer** | vair | who | *Wer ist das?* |
| **wie** | vee | how | *Wie heißt du?* |
| **wo** | voh | where (location) | *Wo wohnst du?* |
| **woher** | vo-HAIR | where from | *Woher kommst du?* |
| **was** | vahs | what | *Was ist das?* |
| **hier** | heer | here | *Was machst du hier?* |
| **machen** | MAH-khen | to do / make | *Was machst du?* |
| **wohnen** | VOH-nen | to live | *Wo wohnst du?* |

---

## 5. Pattern Discovery — W-question word order

**Notice:** Every W-question follows the same shape: **W-word + verb + subject + rest.**

| W | Verb | Subject | Rest |
|---|---|---|---|
| Wer | **ist** | das? | |
| Wie | **heißt** | du? | |
| Wo | **wohnst** | du? | |
| Woher | **kommst** | du? | |
| Was | **machst** | du | hier? |

**Why:** German questions always put the verb **second** — right after the W-word. This is the single most important word-order rule in German. Once the verb is placed, the subject follows immediately, then everything else.

---

## 6. Guided Practice (4 exercises)

### Ex 1 — Recognition — which W-word?
- **Prompt:** *"You want to know a person → ___ ist das?"* → ( Wo ) ( **Wer** ) ( Was )
- **Prompt:** *"You want the location → ___ wohnst du?"* → ( **Wo** ) ( Wer ) ( Wie )

### Ex 2 — Matching
- **Match:** *Wer · Wie · Wo · Woher · Was* ↔ *Who · How · Where (at) · Where from · What*

### Ex 3 — Fill-in (verb position)
- **Prompt:** *"Wo ___ du?" (wohnen)* → ( wohne ) ( **wohnst** ) ( wohnt )

### Ex 4 — Micro-production
- **Prompt:** *Ask: "What are you doing here?"* → **Was machst du hier?**

## Common mistakes
| Mistake | Fix |
|---|---|
| *"Wo du wohnst?"* | Verb second — *Wo **wohnst** du?* |
| *"Woher du kommst?"* | Same rule — *Woher **kommst** du?* |
| Confusing *wo* (location) and *woher* (origin) | *Wo* = at. *Woher* = from. |
