# A1 · Unit 2 · Lesson 2 — Das Verb *sein*
### *Ich bin, du bist…* — Master the full *sein* conjugation

**Level:** A1 · **Unit:** 2 · **Lesson:** 2 of 5
**Capability:** *Use* sein *in all persons to describe yourself and others.*
**Duration:** ~9 min · **New items:** 8 · **Grammar:** *sein* full conjugation (ich bin, du bist, er/sie/es ist, wir sind, ihr seid, sie/Sie sind)
**Spiral:** Nationalities (L1), *ich heiße*, greetings

---

## 1. Learning objective
Learner can:
- **conjugate** *sein* in all six persons,
- **describe** themselves and someone else using *sein*,
- **distinguish** *ihr seid* (you plural, informal) from *Sie sind* (you formal).

## 2. Success criteria
- [x] Unaided — produces *Wir sind fertig. Sie ist Italienerin.*
- [x] Recognises all forms in Mini Review
- [x] Feels "I can use *sein* in every person"

## 5. Pattern Discovery — *sein* full table

| Person | Form | Example |
|---|---|---|
| **ich** (I) | **bin** | Ich bin Deutscher. |
| **du** (you, informal) | **bist** | Bist du fertig? |
| **er / sie / es** (he/she/it) | **ist** | Sie ist Italienerin. |
| **wir** (we) | **sind** | Wir sind neu. |
| **ihr** (you guys) | **seid** | Ihr seid willkommen! |
| **sie / Sie** (they / you formal) | **sind** | Sie sind die Neuen. Sind Sie Herr Klein? |

**Why:** *Sein* is irregular — there's no pattern to memorise the stems. Memorise it as a rhyme: *ich bin, du bist, er ist, wir sind, ihr seid, sie sind.* *Ihr* is the "you guys" form — used when talking to a group of friends. *Sie* (formal) uses the same form as *sie* (they) — context tells you which.

**Emma:** "Don't try to derive *sein* from a rule — just learn it as a song. *Ich bin, du bist, er ist…* Say it out loud five times and it'll stick."

## 6. Guided Practice (4 exercises)

### Ex 1 — Recognition
- **Prompt:** *"Which is 'we are'?"* → ( ihr seid ) ( **wir sind** ) ( sie sind )

### Ex 2 — Matching
- **Match:** *ich bin · du bist · er ist · wir sind · ihr seid · sie sind* ↔ *I am · you are · he is · we are · you guys are · they are*

### Ex 3 — Fill-in
- **Prompt:** *"Wir ___ fertig."* → ( bist ) ( **sind** ) ( seid )
- **Prompt:** *"___ du neu?"* → ( Bin ) ( **Bist** ) ( Ist )

### Ex 4 — Micro-production
- **Prompt:** *Say: She is Italian and we are new.* → **Sie ist Italienerin und wir sind neu.**

## Common mistakes
| Mistake | Fix |
|---|---|
| *"Du sind"* | *du bist* — irregular, just memorise it |
| Confusing *ihr* / *Sie* / *sie* | *ihr* = "you guys" (friends) · *Sie* = "you" (formal, one or many) · *sie* = "they" |
| *"Ihr sind"* | *ihr seid* — the only *seid* form |
