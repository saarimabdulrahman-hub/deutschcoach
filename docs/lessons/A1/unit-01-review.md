# A1 · Unit 1 — Review & Checkpoint
### *Alles zusammen* — Put the whole first unit together

**Level:** A1 · **Unit:** 1 "Hallo!" · **Type:** Unit Review (0 new words)
**Capability:** *Use everything from Unit 1 in one flowing conversation.*
**Duration:** ~7 min · **New items:** 0 (consolidation only) · **Covers:** L1–L6

---

## 1. Learning objective
The learner demonstrates they can **do a complete first meeting** — greet, introduce themselves, ask and answer 3–4 questions, switch register if needed, spell if asked, and say goodbye — all in German, unaided.

## 2. Unit 1 capability checklist
After this review, the learner should comfortably:
- [ ] Greet someone (*Hallo!* / *Guten Tag.*)
- [ ] Say their name (*Ich heiße …*)
- [ ] Ask someone's name (*Wie heißt du?* / *Wie heißen Sie?*)
- [ ] Ask how someone is (*Wie geht's?*)
- [ ] Respond (*Gut, danke. Und dir?*)
- [ ] Say where they're from (*Ich komme aus …*)
- [ ] Say their age (*Ich bin … Jahre alt.*)
- [ ] Choose *du* or *Sie* appropriately
- [ ] Spell their name in German letters
- [ ] Say goodbye (*Tschüss!* / *Auf Wiedersehen.*)

---

## 3. Mixed Review — Interleaved exercises

All items pulled from L1–L6. No new content. Each exercise below represents the final practice block of the review lesson; the full review also includes Recognition, Matching, and Ordering items drawn from the unit pool.

### A — Recognition round (rapid, 5 items)
1. *"Auf Wiedersehen" means?* → Goodbye (formal)
2. *Number 17 in German?* → **siebzehn**
3. *Which letter is 'tset'?* → **Z**
4. *"Wie geht's?" expects what kind of answer?* → **gut / es geht / nicht so gut**
5. *"Woher" means?* → **where from**

### B — Fill-in round (4 items)
1. *Wie ___ du? (name)* → **heißt**
2. *Ich ___ aus Spanien.* → **komme**
3. *Wie ___ Sie? (formal name)* → **heißen**
4. *___ geht's?* → **Wie**

### C — Micro-production (3 items)
1. *Introduce yourself in one sentence.* → **Ich heiße … / Ich bin …**
2. *Ask someone where they're from.* → **Woher kommst du?**
3. *You meet someone in a shop. Greet them.* → **Guten Tag.**

### D — Listen & spell (2 items, audio/script)
1. *(Emma spells "A-N-N-A")* → name: **Anna**
2. *(Emma spells "M-Ü-L-L-E-R")* → name: **Müller**

---

## 4. Speaking Checkpoint (Emma) — the full conversation

**Emma:** "Guten Tag! **Wie heißen Sie?**" → learner replies → "**Freut mich. Woher kommen Sie?**" → learner replies → "**Wie alt sind Sie?**" → learner replies → "**Wie schreibt man Ihren Namen?**" → learner spells → "**Danke! Auf Wiedersehen.**"

*This is the full first-meeting flow from greeting to goodbye — the Unit 1 capability.*
*Learner may use du or Sie throughout — either is fine as long as it's consistent.*

| Grammar patterns | *heißen, sein, haben*, W-questions, *du/Sie*, numbers 0–20, alphabet |

### Unit 1 exit profile
A learner who completes this review can walk into a real first encounter in German and survive it — they can greet, give their name, ask the other person's, check how they are, say where they're from and how old they are, spell if needed, and say goodbye, using the right register. This is the CEFR A1 "Breakthrough" threshold for social interaction.
