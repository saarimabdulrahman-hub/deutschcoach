# A1 · Unit 1 · Lesson 3 — Woher kommst du?
### *Herkunft* — Say where you're from

**Level:** A1 · **Unit:** 1 "Hallo!" · **Lesson:** 3 of 6
**Capability:** *Say where you're from and ask someone else.*
**Duration:** ~9 min · **New items:** 10 · **Grammar pattern:** *kommen* + *aus*; W-question *woher*
**Prerequisites:** L1–L2 (greetings, *ich heiße*, *Wie geht's?*)
**Spiral:** Hallo, ich heiße, danke, Tschüss

---

## 1. Learning objective
Learner can unaided:
- **ask** *Woher kommst du?*,
- **answer** *Ich komme aus …* with ~8 country names,
- **recognise** *Woher* as the "where-from" W-word,
- combine greeting + name + origin into a short self-introduction.

## 2. Success criteria
- [x] Completable unaided
- [x] Produces *"Ich heiße [name]. Ich komme aus [country]."*
- [x] Recognises all 10 items
- [x] Feels "I can say where I'm from"
- [x] Items → SRS

## 4. Vocabulary

## 5. Pattern Discovery — *Woher* + *kommen*

**Notice:** Sofia says *"Ich komme aus Italien."* Ben asks *"Woher kommst du?"*

**Name it: Saying where you're from**
| Who | Form | Example |
|---|---|---|
| ich | **komme** aus … | Ich komme aus Deutschland. |
| du | **kommst** aus … | Woher kommst du? |

**Why:** Same -e/-t switch as *heißen*. You already know this pattern — just swap the verb. *Woher* is the W-word for origin — it asks "from-where".

**Emma:** "You've already seen *ich heiße / du heißt*. *Ich komme / du kommst* is the exact same switch. One pattern, many verbs."

---

## 6. Guided Practice (4 exercises)

### Ex 1 — Recognition
- **Prompt:** *"Welches Land ist 'Germany'?"* → ( Frankreich ) ( **Deutschland** ) ( Spanien )
### Ex 2 — Matching
- **Match** countries German ↔ English: *Italien · Frankreich · Spanien · Österreich* ↔ *Italy · France · Spain · Austria*
### Ex 3 — Ordering
- **Prompt:** → [ aus ] [ komme ] [ Spanien ] [ Ich ] → **Ich komme aus Spanien.**
### Ex 4 — Fill-in (the switch)
- **Prompt:** *"Woher ___ du?"* → ( komme ) ( **kommst** ) ( kommen )
- **Feedback:** "With *du* it's *kommst* — same -t ending as *heißt*."

## Common mistakes
| Mistake | Why | Fix |
|---|---|---|
| *"Ich komme von Deutschland"* | *von* = "from (a person/place)" but *aus* is the standard for countries | Teach *aus* as the country-preposition |
| *"Woher kommen du?"* | Wrong verb form | Pattern Discovery table |
| Forgetting *aus* | "Ich komme Deutschland" | *aus* is mandatory with countries |
