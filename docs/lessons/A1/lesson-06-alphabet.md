# A1 · Unit 1 · Lesson 6 — Das Alphabet
### *Buchstabieren* — Spell your name and recognise German letters

**Level:** A1 · **Unit:** 1 · **Lesson:** 6 of 6
**Capability:** *Spell your name using the German alphabet and understand spelled words.*
**Duration:** ~9 min · **New items:** ~8 (alphabet + the spelling frame; key letters as vocabulary) · **Grammar:** W-question *wie* (extended use: "how do you write that?")
**Spiral:** all U1 greetings, introductions, du/Sie

---

## 1. Learning objective
Learner can:
- **say** the German alphabet,
- **spell** their own name,
- **ask** *Wie schreibt man das?*,
- **understand** spelled names/words (e.g. on the phone),
- recognise that *ß, ä, ö, ü* are extra letters.

## 2. Success criteria
- [x] Can spell own name in German letters
- [x] Asks *Wie schreibt man das?*
- [x] Recognises all 26 letters + ß
- [x] Knows umlauts (ä ö ü) exist

## 5. Pattern Discovery — *Wie schreibt man das?*

**Notice:** Sofia needs to spell her name. The question is *Wie schreibt man das?* — "how does one write that?" *Man* = "one/you" (general, impersonal).

**Why:** *Wie* + verb is the "how" question pattern. You already know *Wie heißt du?* — same structure, different verb. *Man* is a useful impersonal form: *Wie sagt man das?* (How do you say that?).

### Ex 3 — Spelling (listen — script)
- **Prompt (read by Emma/audio):** "Ess · oh · eff · ee · ah" → *What name?* → ( **Sofia** ) ( Sophie ) ( Sara )

## Common mistakes
| Mistake | Fix |
|---|---|
| Spelling *E* as English "ee" / *I* as "eye" | *E* = "ay", *I* = "ee" — reversed from English |
| Forgetting ß is a letter | Treat it like a real letter — it counts |
| *"Wie schreibt du das?"* | *man* is the impersonal form here, not *du* |
