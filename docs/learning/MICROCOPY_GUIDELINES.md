# DeutschFlow — Microcopy Guidelines

**Version:** 1.0 · **Updated:** 2026-07-12 · **Status:** Active · **Sprint:** 6.3A

> The product's voice. Consistent with UX_GUIDELINES writing section and LESSON_WIREFRAMES microcopy. Words are design material, not decoration.

---

## 1. Voice
DeutschFlow sounds like a **calm, intelligent, encouraging coach** — a knowledgeable friend, never a game-show host or a scolding teacher.

| It IS | It is NOT |
|---|---|
| Friendly, warm | Childish, cutesy |
| Calm, unhurried | Urgent, pressuring |
| Intelligent, precise | Vague, filler |
| Encouraging | Flattering / hollow |
| Honest | Manipulative |

**Banned:** sarcasm, guilt ("you're falling behind"), shame, fake urgency, dark patterns, exclamation overload, baby-talk, mascots.

---

## 2. Mechanics
- **Sentence case.** One idea per line. Active voice. Plain verbs.
- Name things by what the learner controls ("review", "lesson"), not system internals.
- Second person ("you"), present tense.
- German sprinkled for immersion where it aids, always with the English near ("Guten Morgen — good morning").
- Emoji: sparingly, only when it clarifies; never as reward confetti in body copy.

---

## 3. Copy library by context

| Context | Copy |
|---|---|
| **Welcome / lesson intro** | "By the end you'll introduce yourself in German." |
| **Warm-up** | "Quick warm-up — you've seen these." |
| **Encouragement** | "Nice — you're getting it." · "Small steps, real progress." |
| **Hint** | "💡 Think about which word means 'am'." |
| **Correct** | "Genau! ✓" · "Perfect." · "You've got it." |
| **Incorrect** | "Almost — the verb comes second. It's *Ich bin Ben*." |
| **Speaking nudge** | "Try it with Emma — mistakes are how you learn." · "No mic? Just type." |
| **Skip** | "You can practice this anytime." |
| **Completion** | "Lesson complete! You can now introduce yourself." |
| **Empty / caught up** | "All caught up — nothing to review right now." |
| **Error (system)** | "Couldn't load this — let's try again." |
| **Exit** | "Leave lesson? Your progress is saved — pick up right here." |

---

## 4. Do / Don't (examples)
| ✅ Do | ❌ Don't |
|---|---|
| "Almost — here's why…" | "Wrong!" |
| "Ready when you are." | "Hurry, your streak is about to break!" |
| "You reviewed 8 words." | "You only did 8 words 😢" |
| "Start your first lesson" | "Begin your epic language quest!!!" |

## 5. Assumptions & trade-offs
- **Assumption:** copy is centralized/reusable so tone stays consistent across hundreds of lessons.
- **Trade-off:** a calm, non-gamified voice is less "sticky" in the dopamine sense than Duolingo's playful nagging — a deliberate brand choice ("serious, not sterile"); we rely on genuine progress + respect for retention rather than guilt loops.
- **Edge case:** localized/German-first copy at higher levels must preserve the same warmth — translate tone, not just words.
